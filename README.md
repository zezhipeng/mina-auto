# 一个微信小程序的脚手架
---

### 安装

```bash
$ npm install -g mina-auto
```

### 使用方法

```bash
$ mina-init webpack <项目名>
```

Example：
```bash
$ mina-init webpack my-mina-project
```

上面的命令会从提取 `mina-webpack` 模板，然后生成 `my-mina-project` 项目
现在模板有 

- [webpack](https://github.com/zezhipeng/mina-webpack)
